Period-over-Period KPI Visualization

Role: Author & Open Source Contributor
Status: Live / Open Source
Year: 2025

Tracking metrics across time periods is critical, but standard BI tools often fall short. I developed this open-source visualization to solve the "cluttered table" problem. It provides a clear, side-by-side comparison of KPIs with intuitive up/down indicators, empowering teams to move from raw data to decisions instantly.

Under the Hood

This project is built as a custom custom visualization plugin, designed to be lightweight and easily integrating into existing data pipelines.


Core Logic: Vanilla JavaScript (`viz-pop.js`) optimized for performance and rendering speed.

Configuration: `manifest.lkml` and `marketplace.json` allow for seamless installation via the Looker Marketplace or manual project import.

Data Handling: Dynamically processes query results to calculate deltas and render conditional formatting on the fly, respecting user-defined thresholds.
