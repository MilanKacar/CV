FlowState: Gamified Recruitment

Role: Product Architect & Full Stack Dev
Duration: Ongoing
Year: 2025

A high-velocity screening platform that solves "Resume Fatigue" by combining AI extraction with a gestural, swipe-based interface. We leverage LLMs to handle the boring data processing, presenting HR managers with clear, scored profile cards. The result? 10x faster screening where AI suggests, but the Human decides.

The Problem with Hiring

Traditional recruitment is broken. AI Automatic Tracking Systems (ATS) are too aggressive, rejecting creative talent because of missing keywords. Conversely, HR managers are drowning in PDFs, spending seconds scanning boring documents and missing hidden gems. This leads to massive "resume fatigue."

The Solution: Make screening as intuitive as a modern dating app, but powered by enterprise-grade data extraction. By transforming dense CVs into visual "Player Cards" we engage the recruiter's intuition. The AI handles the reading; the human handles the judgment.

Under the Hood

This isn't just a UI wrapper; it's a complex data pipeline designed for real-time interaction and zero latency.


Mobile/Front: React Native with Reanimated 2 for silky smooth 60fps gesture animations.

The Brain: Python FastAPI backend serving as the orchestrator.

NLP Engine: OpenAI API (GPT-4o) customized to parse unstructured resume text into standardized skill taxonomies.

Database: PostgreSQL for user data + Redis for caching the "Swipe Stack" to ensure instant card loading.
